Troubleshooting Revapin: Fixes for Common Download Issues

Most Revapin hiccups have a one-step fix. If the toolbar isn't there, refresh the tab. If a download stalled, hit Stop and keep what came through. If your files aren't in Downloads, check where your browser actually puts them. Here's how to clear the handful of snags people run into.

The toolbar isn't showing

Revapin adds a small toolbar to Pinterest pages, and it loads when the page does. If you installed the extension while a Pinterest tab was already open, that tab won't have it yet. Refresh the tab and the toolbar appears. The same fix works if a page has been sitting open for a long time and the extension seems to have dropped off. One reload is almost always enough.

A download stopped partway

Large boards take a while, and sometimes you want to cut a run short or something interrupts it. Hit Stop. The pins that hadn't finished stay selected, so you don't lose your place, and the files that already downloaded are kept. Nothing you'd gathered gets thrown away. When you're ready, start again from where it left off rather than beginning the whole board over.

A big board split into several ZIPs

If a board is large, Revapin breaks the download into more than one ZIP file instead of building one enormous archive that might fail to open. This is expected. Each ZIP holds part of the board, and together they're the complete set. Unzip them all into the same folder and you've got the full board, in order, with nothing missing. A board arriving as three or four archives isn't a fault, it's the tool keeping each file a manageable size.

The files aren't in your Downloads folder

When a download seems to have vanished, it's usually just landed somewhere other than where you looked. Browsers let you set a download location, and some route files to the desktop, a custom folder, or ask each time. A few things to check:

  • Your browser's download settings, to see the folder it saves to;
  • The browser's own downloads list, which links straight to each file;
  • Any "ask where to save" prompt that might be waiting behind another window.

Once you know the folder, the ZIPs are there. It's worth setting a fixed download location so you always know where boards end up.

Still stuck

Revapin is meant to run without much fuss, and the fixes above cover nearly everything. If something's still off after a refresh and a check of your download folder, email support and describe what happened, which board, and what you saw. A real person reads those, and the details help them sort it out faster.
